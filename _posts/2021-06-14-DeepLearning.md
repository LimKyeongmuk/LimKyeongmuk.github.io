# Deep Learning Hardware
### CPU, GPU, TPU
- CPU vs GPU
  - CPU
    - 적은 코어 수
    - 각각의 코어가 더 빠르고, 더 연산력 우월  
    - 연속적인 task에 우월
  - GPU
    - 많은 코어 수
    - 각각의 코어가 느리고 멍청함
    - 병렬적인 task에 우월  
  
- CPU vs GPU in practice
  - 대표적인 CNN 모델들에서 64~76배 성능 차이가 발생할 수 있음
